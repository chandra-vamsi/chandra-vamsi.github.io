# Chandra Vamsi - AI Engineer Portfolio

A premium, modern portfolio built with Next.js 15, React, TypeScript, Tailwind CSS, and Framer Motion. 
Designed for an AI Engineer working with Python, Generative AI, RAG, and Machine Learning.

## Tech Stack
- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- shadcn/ui

## Getting Started Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment to GitHub Pages

This repository is configured to automatically deploy to GitHub Pages when changes are pushed to the `main` or `master` branch.
The deployment is handled via the `.github/workflows/deploy.yml` GitHub Actions workflow.

Ensure that **GitHub Pages** is enabled in your repository settings:
1. Go to **Settings** > **Pages**.
2. Under **Build and deployment**, select **GitHub Actions** as the source.
